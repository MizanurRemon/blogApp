# blogApp

##Description

A simple android app for uploading data into Firebase and fetching data from firebase with email registration and sign in authentication with email and password.

###Tools

* Relative Layout
* Linear Layout
* Cardview
* Recyclerview
* Splashscreen
* Firebase
* FirebaseAuth
